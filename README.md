# ts-promise-from-functor-observer

Typescript version Promise.

## 使用

1.  将代码 clone 到本地
2.  测试

    1.  在命令行中依次输入

    ```shell
    npm install
    npm run build
    npm test(:es5|:es6|:ts)
    ```

    进行 Promise/A+ 测试

3.  调试

    1.  在命令行中依次输入

    ```shell
    npm install
    npm run dev
    ```

    进行调试
